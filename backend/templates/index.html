❓ Help

How to use Voxa AI:

  • Click + New Chat to start a fresh conversation
  • Your chats are saved automatically and shown in the sidebar
  • Ask anything
  • Click any past chat to continue where you left off
  • Click ✕ on a chat to delete it permanently

Tips:

  • Try: "What is DSA?"
  • Try: "Who made you?"
  • Try: "What is AI??"

ℹ️ About Voxa AI

Voxa AI is an intelligent assistant built by Chandan Singh.

It is powered by:

  • LangGraph — agent orchestration & tool calling
  • OpenRouter — access to free LLMs
  • RAG (ChromaDB) — answers from a custom knowledge base
  • FastAPI — backend server
  • JWT Auth — secure login & saved chat history

Built with ❤️ using Agentic AI concepts.
